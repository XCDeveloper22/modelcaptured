# Desktop & Mobile App Installation Guide

## Current Setup
The application is configured as a **Progressive Web App (PWA)** that can be installed on:
- **Desktop**: Windows, macOS, Linux (via browser install prompt)
- **Mobile**: Android, iOS (home screen installation)

## Building Desktop Installers

### Option 1: Electron-based Desktop App (.exe, .dmg)
To create native installers, install Electron and electron-builder:

```bash
npm install --save-dev electron electron-builder
```

Create `electron-main.js`:
```javascript
const { app, BrowserWindow } = require('electron');
const isDev = require('electron-is-dev');
const path = require('path');

let mainWindow;

app.on('ready', () => {
  mainWindow = new BrowserWindow({
    width: 1200,
    height: 800,
    webPreferences: { preload: path.join(__dirname, 'preload.js') }
  });

  const startUrl = isDev ? 'http://localhost:5173' : `file://${path.join(__dirname, '../dist/index.html')}`;
  mainWindow.loadURL(startUrl);
});
```

Update `package.json`:
```json
{
  "main": "electron-main.js",
  "homepage": "./",
  "build": {
    "appId": "com.modelcaptured.app",
    "productName": "Model Captured",
    "files": ["dist/**/*", "node_modules/**/*"],
    "win": {
      "target": ["nsis", "portable"]
    },
    "mac": {
      "target": ["dmg", "zip"]
    }
  },
  "scripts": {
    "build:exe": "npm run build && electron-builder --win",
    "build:mac": "npm run build && electron-builder --mac"
  }
}
```

Build installers:
```bash
npm run build:exe   # Creates .exe for Windows
npm run build:mac   # Creates .dmg for macOS
```

### Option 2: Current PWA Installation
Users can install via:
1. **Desktop**: Click the "Install App" button in Hero section
2. **Mobile**: Click "Install on Mobile" in Contact section
3. The app will be added to home screen/taskbar

## File Structure
```
public/
  manifest.json          # PWA configuration
  model-captured-installer.exe   # (To be generated by electron-builder)
  model-captured-installer.dmg   # (To be generated by electron-builder)

index.html             # PWA meta tags already configured
src/App.tsx           # Install buttons in Hero & Contact sections
```

## Testing PWA Installation
1. Build: `npm run build`
2. Preview: `npm run preview`
3. Look for install prompt in desktop browsers (Chrome, Edge)
4. Mobile: Add to home screen via browser menu
